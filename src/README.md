## Getting Started

## Pré-requisitos

- Docker
- Docker-compose

## Executando a aplicação

crie a network configurada no docker-compose.yml (websocket-net) usando o seguinte comando:

```
docker network create "websocket-net"
```

rode o seguinte comando no terminal:

```
docker-compose up
```
